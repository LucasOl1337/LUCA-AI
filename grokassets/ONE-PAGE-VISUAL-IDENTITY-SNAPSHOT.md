# LUCA-AI (LUC.AI) — One-Page Visual Identity Snapshot

**Project:** LUCA-AI (LUC.AI)
**Date:** 2026-05-31

## Core Metaphor
Wes Anderson-style miniature command center / diorama inside a sophisticated AI agent laboratory. Agents as small, perfectly composed mechanical models performing missions on a central canvas.

## Primary Colors
- Admiralty Navy: #0B1F3A (dominant dark)
- Royal Fleet Blue: #1E4E8C
- Officer White: #F8FAFC (ceremonial bright)
- Sophisticated pastels for accents on dark tech backgrounds

## Typography Direction
Clean, slightly playful but professional sans (Inter spirit). Symmetry and alignment are paramount. Small mechanical labels in mono for terminals and status.

## Key Visual Motifs
- Symmetrical miniature dioramas and mechanical models
- Agent cards as tiny worlds
- Central canvas / stage as command center
- Physical button feel, precise transitions
- Cinematic composition, high-end operational delight

Meticulous symmetry, charming miniature laboratory aesthetic, delightful precision. Navy + pastel sophistication meets high-agency AI operations. Not cyberpunk overload, not generic dashboard — elegant, cinematic, Wes Anderson-inspired command center for serious agent work.
